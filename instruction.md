# Подсказки по командной строке
# (размещение на удаленный репозиторий CanUFeelMyHeart)

Команда смены
директории
```sh
cd c:\folder_name
```

команда отображения текущей директории
```sh
pwd
```

Листинг текущей директории
Windows:
```sh
dir
```
Linux, MacOS:
```sh
ls
```

Удаление файла в Windows:
```sh
del <filename>
```

в Linux, MacOS:
```sh
rm <filename>
```
